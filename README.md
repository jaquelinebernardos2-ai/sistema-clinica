# Sistema de Atendimento para Clínicas Médicas

## Descrição
Este projeto foi desenvolvido com o objetivo de informatizar o agendamento de consultas em uma clínica médica.

## Funcionalidades
- Cadastro de usuários
- Login com autenticação
- Agendamento de consultas
- Consulta de endereço via CEP
- Simulação de previsão do tempo

## Tecnologias utilizadas
- Node.js
- Express
- JavaScript

## Como executar
1. Instalar as dependências:
npm install

2. Executar o servidor:
node server.js

3. Abrir o arquivo index.htl no navegador

## Observações
A funcionalidade de clima foi simulada para representar a integração com API externa.
